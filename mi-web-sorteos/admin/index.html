Mercado Pago — PIX Real

Cole seu Access Token de produção ou teste do Painel Mercado Pago. Com o token configurado, os pagamentos PIX são gerados e confirmados automaticamente.

⚠ Importante — dois tipos de token:

TEST-... → Credenciais de teste. Use no localhost para simular PIX sem cobrar dinheiro real.

APP_USR-... → Credenciais de produção. Só funciona com site publicado em HTTPS (domínio público). Não funciona em localhost.

Detectada: —

Onde copiar (não confunda):
✓ Access token em Credenciais de teste → começa com TEST-
✗ TESTUSER... = usuário de teste (comprador/vendedor) — não serve aqui
✗ Public Key — também não é o Access token

Webhook: —

Como funciona:
1. Mercado Pago Developers → sua aplicação
2. Menu lateral: Credenciais de teste
3. Copie só o campo Access token (TEST-... longo)
4. Cole no campo "Token de TESTE" acima e salve
